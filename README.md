# 악보이미지 → MusicXML 변환기

악보 이미지(스캔/촬영)를 입력으로 받아 MusicXML 파일로 변환하는 프로젝트입니다.

## 주요 기능
- 악보 이미지 입력 지원
- MusicXML 파일 출력

## 개발 환경
- Python 3.x
- 가상환경(venv) 사용

## 설치 및 실행
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```

## 사용 방법
1) 변환할 악보 이미지를 준비합니다.
2) 프로젝트 실행 후 안내에 따라 변환합니다.
3) 출력된 MusicXML 파일을 확인합니다.

## 디렉터리 구조
- `main.py`: 변환 실행 스크립트
- `main2.py`: 보조/실험용 스크립트
- `requirements.txt`: 의존성 목록
